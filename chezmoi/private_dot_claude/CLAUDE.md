# Global Preferences

## Coding Style
- Prefer explicit over implicit
- Keep solutions simple — don't over-engineer
- Only add comments where logic isn't self-evident

## Environment
- Dotfiles managed with Nix (nix-darwin + home-manager) and Chezmoi
- Machines: macOS (behemoth), Linux (rocinante, stargazer)
- Shell: nushell

## Tools
- Use nix for package management, not brew (except GUI casks on macOS)
